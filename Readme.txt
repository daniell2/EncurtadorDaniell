Encurtador de links feito em Django

Requisitos:

Python 2.6
Django

Como rodar:

- Dezipe o arquivo baixado para o C:\

- renomeie a pasta daniell2-Encurtador.....
  para EncurtadorDaniell


- Va para o prompt do MS-DOS


- No prompt va ate a pasta do projeto e digite:

  python manage.py runserver


- No browser digite:

  localhost:8000
